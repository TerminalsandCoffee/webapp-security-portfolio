# Security Incident Triage Assistant

Planned incident-response project for turning security alerts and log bundles into concise investigation notes and next steps.

## Goal

Build a tool that can:

- ingest alerts, WAF logs, and supporting context
- identify probable attack type and affected surface
- assign severity and explain likely customer impact
- recommend triage, containment, and remediation steps
- generate a short summary for operators and a clear summary for customers

## Why It Fits This Portfolio

This project is intended to demonstrate incident handling, troubleshooting, and communication workflows that align with cloud security operations and enterprise services work.

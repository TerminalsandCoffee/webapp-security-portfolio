# Smart WAF Analyzer

Planned analytics project for reviewing WAF logs and turning raw events into actionable recommendations.

## Goal

Build a tool that can:

- ingest WAF logs from JSON or CSV exports
- summarize attacks by type, rule, IP, URI, and geography
- highlight likely false positives and noisy rules
- recommend tuning changes and rollout guidance
- produce analyst-ready and customer-friendly summaries

## Why It Fits This Portfolio

This project is intended to demonstrate the monitoring, advanced reporting, and operational guidance side of cloud application security engineering.
